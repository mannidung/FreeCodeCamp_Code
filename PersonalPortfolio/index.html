Welcome

This is me, Pär Persson Mattsson, a support engineer and HPC consultant at a software company during the day, and a student of Free Code Camp during the night.

On this page you will find links to my projects, both to those made as a part of the Free Code Camp course, and to other things that I have done or am currently working on.

You can find the code of my projects at either Github or BitBucket. Remember to check back every now and then to see what new projects I am working on.

Personal Portfolio

This website!
It was created with the following goals in mind:
  • All of the portfolio webpage's content should be accessible by just scrolling.
  • There should be clickable buttons, leading to my different social media pages.
  • There should be thumbnail images of different projects.
  • It should be possible to navigate to different sections through buttons in a navigation bar.


Technologies/Libraries used:

Javascript, CSS, Bootstrap, Font Awesome
